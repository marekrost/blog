---
title: "Nový core server ve WebMedea"
date: 2018-06-06
slug: novy-core-server-ve-webmedea
---

Protože naše databáze se neustále rozšiřuje, a my k ní navíc přidáváme další funkce, museli jsme ve [WebMedea Services](https://www.webmedea-services.cz) rozšířit i náš hardware o další hlavní server. Server se bude starat o výpočty důležitosti domén na internetu a o aktuálnost odkazů mezi těmito doménami.
